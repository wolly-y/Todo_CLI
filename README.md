# Менеджер задач (Todo CLI)

Консольное приложение для управления списком задач.
Данные хранятся в SQLite. Можно добавлять, редактировать, 
удалять задачи, отслеживать статусы и дедлайны.

## Возможности

- Добавление задач с названием, описанием, приоритетом и дедлайном
- Просмотр всех задач с фильтрацией по статусу и сортировкой
- Изменение статуса задач (новая -> в работе -> сделано)
- Редактирование описание и приоритета
- Удаление задач с подтверждением
- Поиск просроченных задач
- Статистика по статусам и приоритетам

## Технологии

- Python 3.14.2
- SQLite3 (встроенная библиотека)
- pytest (для тестирования)

## Структура проекта

```
todo-cli/
├── database.py      # Работа с базой данных SQLite
├── models.py        # Бизнес-логика и валидация
├── main.py          # Консольное приложение
├── test_database.py # Тесты для database.py
└── README.md
```

## Установка и запуск

1. Склонируйте репозиторий:
   ```bash
   git clone https://github.com/wolly-y/Todo_CLI.git
   cd Todo_CLI

2. Запустите приложение:
    ```
   python main.py

3. Для запуска тестов:
    ```
   pip install pytest
   pytest test_database.py -v

## Пример использования

```
========================================
МЕНЕДЖЕР ЗАДАЧ
========================================
1. Добавить задачу
2. Показать все задачи
3. Изменить статус задачи
4. Редактировать задачу
5. Удалить задачу
6. Просроченные задачи
7. Статистика
8. Выход
========================================
Выберите действие (1-8): 1
Название: Купить молоко
Описание (можно оставить пустым): В магазине у дома
Приоритет (высокий, низкий, средний):
> высокий
Дедлайн (ГГГГ-ММ-ДД, можно оставить пустым): 2026-06-01
Задача создана. ID: 1
```

## Примечания

- Приоритет по умолчанию: ```средний```
- Дедлайн по умолчанию: ```Без дедлайна```
- Название задачи должно быть уникальным
- База данных создается автоматически при первом запуске
